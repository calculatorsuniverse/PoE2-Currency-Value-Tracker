## PoE2 Currency Hierarchy

| Currency | Primary Use | Relative Value |
|----------|------------|---------------|
| Divine Orb | Reroll explicit modifier values | Highest (reference) |
| Chaos Orb | Reroll rare item modifiers | ~50–100 per Divine |
| Exalted Orb | Add modifier to rare item | ~10–20 per Divine |
| Orb of Annulment | Remove random modifier | ~5–15 per Divine |
| Orb of Alteration | Reroll magic item modifiers | Low |
| Orb of Transmutation | Upgrade normal to magic | Very low |
| Orb of Augmentation | Add modifier to magic item | Very low |
| Mirror of Kalandra | Duplicate any non-unique item | Rarest / Highest |

*Exchange rates fluctuate based on league economy and patch changes. Always check poe2.trade for live rates.*
